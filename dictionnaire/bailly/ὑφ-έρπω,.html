ὑφ-έρπω,   se glisser sous ou dans : τρόμος μ’ ὑφέρπει, Eschl. Ch. 464, le frisson se glisse dans mes membres ; cf. Eschl. Ag. 270 ; PhilstR. p. 500 ; ou, avec le dat. (avec tmèse) φθονερὸν δ’ ὑπ’ ἄλγος ἕρπει Ἀτρείδαις, Eschl. Ag. 450, une indignation sourde envahit le cœur contre les Atrides ; cf. Jos. B.J. 5, 8, 2.
