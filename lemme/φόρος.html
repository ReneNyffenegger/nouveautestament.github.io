φόρος (phoros)

impôt, impôts


impôt, charge, taxe, tribut, le tribut, spécialement la taxe levée sur les maisons, les terres et les personnes
Nom masculin
Vient de φέρω

φόρος de φέρω, une charge (comme porté), c-à-d (au sens figuré) une taxe (correctement, une évaluation individuelle sur les personnes ou la propriété, alors que τέλος est d'habitude un péage général sur les marchandises ou le voyage) : hommage.

φόρος : Anglais : that which is brought in, tribute -|- Français : ﻿ce qui est apporté, hommage
