στηλ-όω, Dor. στᾱλόω, set up as a στήλη or monument, πέτρον στάλωσε ἐπ’ ἠρίῳ AP7.394 (Phil.); τάφον Epigr. in POxy.662.28 (Amyntas); σωρὸν λίθων ἐπί τινα LXX2 Ki.18.17:—Pass., to be so set up, stand firm, ib.Jd.18.16, etc.:—Med., στηλοῦσθαί τινι devote oneself to another, Suid.     2. inscribe on a στήλη, OGI221.15 (Ilium, iii B.C.):—Pass., Milet.6.36 (Didyma).     3. mark out with boundary pillars, τὴν χώραν OGI225.30 (Didyma, iii B.C.), cf. 335.74 (Pergam., ii B.C.).
