κουρεῶτις,ιδος   [ ῐδ ] adj. f. κ. ἡμέρα ou ἑορτή, Plat. Tim. 21 b ; AlciphR. 3, 46, 1, jour ou fête de la tonte, le 3 e jour des Apaturies, où l’on amenait, dans l’assemblée des φράτορες, les jeunes enfants de 3 ou 4 ans pour les faire inscrire sur le registre public ( κοινὸν γραμματεῖον ) ; cette inscription était accom- pagnée du sacrifice d’un agneau ou d’une brebis ; sel. d’autres, le nom de cette fête venait de ce qu’on coupait pour la 1 ʳe fois ce jour-là les cheveux des jeunes enfants ( κουρά ).
