ἀνα-ζεύγνυμι   (impf. ἀνεζεύγνυν, f. inus., ao. ἀνέζευξα, pf. inus.) litt. atteler (les chevaux, les bêtes de somme) pour le retour, d’où ἀ. στρατόν, Hdt. 9, 41, ramener une armée ; νῆας, Hdt. 8, 60, ramener une flotte ; στρατόπεδον, Hdt. 9, 58, lever un camp ; abs. ἀν. lever le camp, Thc. 8, 108 ; XÉn. An. 3, 4, 37, etc. ; d’où revenir, en parl. d’une armée, Plut. Ant. 84.
