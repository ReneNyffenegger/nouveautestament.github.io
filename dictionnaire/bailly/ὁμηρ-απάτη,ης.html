ὁμηρ-απάτη,ης   ( ἡ ) [ ᾰᾰ ] récit mensonger d’Homère ; ὁμηραπάτης ἐπικόπτης, Timon 40, 1 (Sext. P. 1, 224, p. 58) ; DL. 9, 18, qui se moque des récits mensongers d’Homère sur les dieux, ép. de Xénophanès ( Ὅμηρος, ἀπάτη ; sel. d’autres, moins bien, Ὁμηροπάτης, qui foule aux pieds Homère, de Ὅμ. πατέω ).
