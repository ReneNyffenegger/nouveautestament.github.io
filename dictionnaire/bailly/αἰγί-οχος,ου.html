αἰγί-οχος,ου   ( ὁ ) qui tient ou brandit l’égide (Zeus) Il. 2, 375, etc. ; Od. 9, 275, etc. || → Gén. dor. -όχω, ThcR. Idyl. 26, 31 ( αἰγίς ; p. le second terme, cf. γαιήοχος ; sel. d’autres, le sens est « qui passe rapidement », cf. αἰγίς I ; ou encore « qui conduit la nuée orageuse », où αἰξ serait le nom d’un oiseau annonciateur d’une tempête, cf. all. Himmelsziege).
