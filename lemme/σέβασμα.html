σέβασμα (sebasma)

objet de vénération, objets de vénérations

MorphologieNom neutre
OrigineVient de σεβάζομαι
σεβάζομαιfaire vénération
Définition Aobjet de vénération, tout ce qui est honoré religieusement, objet d'adoration, de culte. de temples, autels, statues, images idolâtres
Définition Bσέβασμα de σεβάζομαι, quelque chose d'adoré, c-à-d un objet de vénération (Dieu, autel, etc.) : la dévotion, qui est vénérée.
Définition Cσέβασμα : Anglais : that for which awe is felt, an object of awe -|- Français : ﻿ce pour quoi la crainte est ressentie, un objet de crainte
σέβασμα nom sg neut acc
Définition Dσέβασμα, ατος, τό : Nom, Neuter
EN 1 : an object of worship

FR 1 : un objet de culte
FR 2 : un objet d'adoration ou de vénération.
