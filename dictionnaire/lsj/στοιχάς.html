στοιχάς, άδος, ὁ, ἡ, (στοῖχος) in a row one behind another, esp. αἱ Στοιχάδες (sc. νῆσοι), name of the islands which lie in a row east of Toulon, now les Îles d' Hyères, A.R.4.554, Str.4.1.10.     2. ἐλᾶαι σ. olive-trees (prob. because planted in rows) which were not sacred, like the μορίαι, Sol. ap. Poll.5.36, Philoch.62.     II. στοιχάς, ἡ, an aromatic plant, cassidony, Lavandula Stoechas, Orph.A.918, Dsc.3.26.
