ζωγρεία,ας   (ἡ) capture d’un prisonnier vivant, Pol. 1, 15, 2, etc. ; StR. 359 ; ζωγρείᾳ ἁλῶναι, Pol. 5, 86, 5, être pris vivant ; ζωγρείᾳ ἐγκρατῆ γίγνεσθαί τινος, Pol. 1, 9, 8 ; ou ζωγρείᾳ κύριον γίγνεσθαί τινος, Pol. 1, 17, 4, se rendre maître de qqn ou le capturer vivant ( ζωγρεύς ; cf. ζωγρία ).
