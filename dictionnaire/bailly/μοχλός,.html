μοχλός,   οῦ (ὁ) barre de bois, d’où : 1 levier pour déplacer un navire, Od. 5, 261 ; pour forcer une porte, EuR. Or. 1474, etc. ; pour soulever un fardeau, AR. Pax 307 || 2 pieu, Od. 9, 332 ; EuR. Cycl. 633 || 3 barre de bois assujettie au moyen du βάλανος pour fermer une porte, d’où verrou, Thc. 4, 111 ; AR. Vesp. 112, etc. ; τὸν μοχλὸν ἐμβάλλειν, XÉn. An. 7, 1, 12, mettre le verrou ; πύλας μοχλοῖς χαλᾶν, Eschl. Ch. 879, ouvrir la porte en retirant le verrou ; fig. en parl. du gardien d’une maison, Soph. (Ath. 99 d) (pré-grec).
