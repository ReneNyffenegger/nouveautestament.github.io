Crat. 426 c, si tu veux ; cf. Plat. Conv. 210 a, etc. ; Thc. 4, 46 ; 6, 13, etc. ; postér. avec l’ind. Pol. 9, 31, 2 ; rar. après un mot : δύ’ ἂν ὦσι, DÉm. s’ils sont deux ; chez les Trag. seul. dans la locut. : οὐδ’ ἄν, pas même si, Soph. O.R. 1062 dout. ; se joint à d’autres particules : ἂν καί, même si, quand bien même, Plat. Soph. 230 d, Leg. 859 d ; ἂν μόνον, si seulement, pourvu que, Plat. Theæt. 172 d, Pol. 293 b ; ἄν περ, si toutefois, Plat. Tim. 90 c, Leg. 631 c, etc. ; ἂν... τε, ἂν... τε, soit… soit, Plat. Gorg. 512 a, Pol. 259 b, etc. ; ἄν τε... καὶ ἐάν, Plat. Pol. 292 e ; ou ἂν... καὶ ἄν, Plat. Crat. 385 c, m. sign. || 2 dans le disc. indir. avec le sbj. σκόπει... ἄν πῃ δυνώμεθα, Plat.
