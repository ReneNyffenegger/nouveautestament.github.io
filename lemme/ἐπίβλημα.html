ἐπίβλημα (epiblêma)

surjet


ajout, ce qui est posé ou rajouté sur une chose, une addition, une pièce
Nom neutre
Vient de ἐπιβάλλω

ἐπίβλημα de ἐπιβάλλω, une pièce : morceau.

ἐπίβλημα : Anglais : that which is thrown over, covering -|- Français : ﻿ce qui est renversé, couvrant
ἐπίβλημα nom sg neut acc

ἐπίβλημα, ατος, τό : Nom, Neuter
EN 1 : that which is put on, a patch
EN 2 : a patch on a garment.
FR 1 : ce qui est mis, un patch
FR 2 : un patch sur un vêtement.
