Prot. 322 b, ils se faisaient tort les uns aux autres ; πρὸς αὑτὰ καὶ πρὸς ἄλληλα, Plat. Gorg. 451 c, les choses considérées en elles-mêmes et dans leurs rapports les unes avec les autres ; cf. Il. 4, 62, etc. ; Od. 1, 209, etc. ; Eschl. Pr. 200, etc. ; Thc. 8, 66 ; Plat. Tim. 38 d, 54 b ; Prot. 322 b, etc. ; ὁ δι’ ἀλλήλων τρόπος, DL. 9, 89 ; ou ὅρος, Nicom.
