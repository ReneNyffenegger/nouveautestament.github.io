( τὰ ) [ πᾰᾰθ ] les Panathénées, fêtes athé- niennes en l’honneur d’Athèna, fondées par Érikhthonios sous le nom d’ Ἀθήναια (IstR. fr. 3) et réorganisées par Thésée sous le nouveau nom. On distinguait les grandes Panathénées qui se célé- braient tous les quatre ans, la 3 e année de l’Olympiade, le 28 Hé- katombæon, et les petites qui étaient annuelles, ou sel. d’autres, triennales, Hdt. 5, 56 ; Thc. 5, 47 ; XÉn. Conv. 1, 2 ; IsocR. 236 c, etc. ( π. Ἀθηναῖος ; cf. Παλλάς ).
