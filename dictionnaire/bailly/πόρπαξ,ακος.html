πόρπαξ,ακος   ( ὁ ) [ ᾱκ ] 1 anneau ou poignée de bouclier dans lesquels on passe le bras, Bacchyl. fr. 4, 69 ; Soph. Aj. 576 ; EuR. Ph. 1127 ; on pouvait démonter cette poignée, d’où ἔχουσι πόρπακας ( αἱ ἀσπίδες ) AR. Eq. 858, les boucliers ont leur poignée, c. à d. sont prêts || 2 p. suite, le bouclier lui-même, AR. Eq. 1372 ( πόρπη ).
